Generate SEO-optimised alt text for your images using AI. Bulk process your entire media library in clicks.

== Description ==

Alt Text AI uses advanced vision AI to analyse your images and generate accurate, SEO-friendly alt text automatically.

**Features:**

* One-click alt text generation for individual images
* Bulk generate alt text for your entire media library
* Industry-specific context (roofing, dental, electrical, etc.)
* Location-aware for local SEO
* 125-character optimised alt text following SEO best practices
* Works with the WordPress media library — no complex setup

**How It Works:**

1. Install and activate the plugin
2. Enter your license key or start a free trial
3. Set your industry and location in Settings → Alt Text AI
4. Click "Generate with AI" on any image, or bulk process from the media library

== Installation ==

1. Upload the `alt-text-ai` folder to `/wp-content/plugins/`
2. Activate the plugin through the Plugins menu
3. Go to Settings → Alt Text AI to configure

== Frequently Asked Questions ==

= How many images can I process? =

Depends on your plan: Starter (100/month), Pro (500/month), or Agency (2,000/month). A 14-day free trial includes 25 images.

= Does it auto-generate alt text on upload? =

Not yet — this is planned for v1.1. Currently you generate manually or via bulk action.

= What AI model is used? =

We use OpenAI's GPT-4o-mini vision model for fast, accurate image analysis.
